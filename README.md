# freeBSD_lab1
## Завдання
Створіть функцію, яка приймає вхідний файл у форматі CSV та виводить його вміст у вигляді відформатованої таблиці. Додайте підтримку різних роздільників, таких як кома, крапка з комою або табуляція.
